When transferring cryptocurrency (crypto), you should only send it to the correct wallet. Sending crypto to the wrong address is irreversible and the funds cannot be recovered. You should also ensure that the sending and receiving wallets are compatible, *especially when dealing with different types of cryptocurrencies. For example, you should only send ETH to Ethereum wallets or BTC to Bitcoin wallets.

Here are some tips for transferring crypto:

  • Triple-check the recipient's address.
  • Copy and paste the address to ensure it's entered correctly.
  • Check the current state of the network before sending.
  • Monitor the transaction until it has been fully confirmed.
  • Only share the link with the intended recipient.
